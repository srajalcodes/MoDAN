# Model Availability

The pretrained MoDAN model checkpoint is publicly available through the Zenodo reproducibility archive:

**Zenodo DOI:** https://doi.org/10.5281/zenodo.21221081

The archive includes:

- `best_biomodal_model.pt` (pretrained MoDAN checkpoint)
- ChemBERTa embeddings
- ESM-2 embeddings
- BioBERT embeddings
- DrugBank evaluation splits
- BIOSNAP benchmark splits
- ZhangDDI benchmark splits

Researchers can download the archive and extract it into the project directory before running evaluation or transfer learning experiments.
