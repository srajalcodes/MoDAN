# MoDAN

Modality-Disentangled Attention Network (MoDAN) for zero-shot drug-drug interaction prediction.

## Modalities

- ChemBERTa embeddings
- ESM-2 embeddings
- BioBERT pathway embeddings

## Training Dataset

DrugBank v5.1.21

## Intended Use

Prediction of binary drug-drug interactions under strict inductive settings.
